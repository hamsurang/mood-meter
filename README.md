## 무드미터가 뭐예요?

무드미터는 예일대학교 감성지능센터에서 만든 **감정 인식 도구**예요. 가로축은 쾌적함, 세로축은 에너지 수준을 나타내서, 100가지 감정 단어 중 지금 내 기분에 가장 가까운 걸 골라볼 수 있어요.

막연하게 "기분이 별로야"라고 느꼈던 감정에 이름을 붙여주면, 그것만으로도 마음이 한결 가벼워지더라고요.

## 사용법

1. **10x10 그리드**에서 지금 기분에 가장 가까운 감정을 찾아 눌러요
2. 짧은 **한마디**를 적어요 (안 적어도 괜찮아요)
3. **복사하기** 버튼을 누르면 이미지가 클립보드에 복사돼요. 디스코드나 슬랙에 바로 붙여넣기 하세요!

## 이야기

함수랑(hamsurang) 커뮤니티에서 [@sonsurim](https://github.com/sonsurim) 님이 공유해 준 무드미터를 [@minsoo-web](https://github.com/minsoo-web) 님이 받아서 커뮤니티에 전파해 주셨어요. 좋은 것을 나누는 Giver 마인드가 이어져서, 이번엔 웹앱으로 만들어 봤어요.

## 개발하기

```bash
git clone https://github.com/hamsurang/mood-meter.git
cd mood-meter
pnpm install
pnpm dev
```
